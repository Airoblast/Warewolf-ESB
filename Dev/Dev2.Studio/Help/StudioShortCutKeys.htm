Studio Short Cut Keys

A fully comprehensive look at short cut keys in the Studio.

Generally we use ALT for opening or focusing on areas of the system like the Connection Manager area, or the New Workflow wizard. Typically if that area is already open, then the shortcut will bring that area into focus in the Work Environment.

Ctrl is typically used for executing tasks like Save, Cut, Undo etc.

ALT

Alt + C → Connection Manager tab

Alt + N → New Service Wizard (Result of Wizard saves new service)

Alt + B → Backup Manager tab

Alt + M → Community tab

Alt + S → Schedule Manager tab

Alt + A → App Store tab

Alt + L → Licence Manager tab

Alt + W → Web Server Manager tab

Alt + H → Home Page tab

Alt + U → Update Manager tab

Alt + P → People and Security Manager tab

Alt + T → Settings Manager tab

Alt + F → Feedback window

Alt + A → Open All Management tabs that are not open

The control short cuts are context based. If the context of the page is say Settings, then CTRL + R (Run or Execute) is meaningless and will do nothing.

CTRL

Ctrl + S → Save

Ctrl + R → Run (Execute)

Ctrl + D → Deploy

Ctrl + T → Debug Test

Ctrl + I → Info (version, created date, created by etc.)

Ctrl + X → Cut

Ctrl + C → Copy

Ctrl + V → Paste

Ctrl + Z → Undo

Ctrl + Y → Redo

Ctrl + B → Check Build (Checks if something will compile or not)

Ctrl + N → New Database Wizard

CTRL + SHIFT

Ctrl + Shift + S → Save All

Ctrl + Shift + F4 → Close All tabs (Offer a save option for tabs that require it)

Ctrl + Shift + D → Deploy All

Ctrl + Shift + B → Check Build All
